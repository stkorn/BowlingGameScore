# BowlingGameScore
Bowling frameScore calculator

::: How to play :::

1. Enter the sequence of roll (any length of sequence of character) in console or commandline

   input character includes,
   • 0 - 9 is number of pin down
   • X  is a Strike
   • /  is a Spare 
   • -  is a Miss
   
2. Press enter
3. See your scores!
